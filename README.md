🎬 Netflix Data Analysis Project
📌 Overview

This project performs Exploratory Data Analysis (EDA) on a Netflix dataset using Python. The goal is to analyze Netflix content trends, clean missing data, and visualize important insights about Movies and TV Shows available on the platform.

The project uses Pandas, NumPy, Matplotlib, and Seaborn for data analysis and visualization.

🚀 Features
Dataset loading and inspection
Data cleaning and preprocessing
Handling missing values
Movies vs TV Shows analysis
Release year trend visualization
Basic statistical insights
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
