Netflix Data Analysis 📊

A simple exploratory data analysis (EDA) project on Netflix dataset using Python, Pandas, and Matplotlib.
This project focuses on cleaning missing data and visualizing trends in Netflix content such as Movies vs TV Shows and release year distribution.

🚀 Features
Load and inspect Netflix dataset
Handle missing values
Analyze dataset structure
Visualize:
Movies vs TV Shows
Content release trends over years
🛠️ Technologies Used
Python
NumPy
Pandas
Matplotlib
Seaborn
Jupyter Notebook

├── Netflix_EDA.ipynb
